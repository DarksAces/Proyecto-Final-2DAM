# Object Capture 3D Mobile App

Flutter application for capturing objects and viewing 3D models.

## Setup

1. Install Flutter dependencies:
   ```bash
   flutter pub get
   ```

2. Configure API URL:
   Edit `.env` and set `API_URL` to your backend URL.
   - For Android Emulator: `http://10.0.2.2:8000`
   - For Physical Device: `http://YOUR_LOCAL_IP:8000`

3. Run the app:
   ```bash
   flutter run
   ```

## Permissions

Ensure you grant camera permissions when prompted.
