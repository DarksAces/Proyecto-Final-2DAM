# Object Capture 3D

A full-stack application to capture real-world objects and transform them into 3D models.

## Structure

- `backend/`: Python FastAPI server for 3D processing.
- `mobile/`: Flutter mobile application for capturing and viewing.

## Quick Start

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `uvicorn app.main:app --reload`

### Mobile
1. `cd mobile`
2. `flutter pub get`
3. `flutter run`

See `backend/README.md` and `mobile/README.md` for more details.
